# Messenger
Веб-приложение для видеозвонков
## Зависимости
| Название                                               | Предназначение |
|:-------------------------------------------------------|:---------------|
| [Network](https://github.com/idDeqpt/Network)          | Web сервер     |
| [CLI](https://github.com/idDeqpt/CommandLineInterface) | Взаимодействие с аргументами командной строки |
____
# Требования
- CMake 3.14+
- C++17-совместимый компилятор
- npm
- Наличие TLS сертификатов для защищённого режима

Формат сертификатов следующий:
- full_certificate_chain.pem - файл с полной цепочкой сертификатов
- private_key.pem - файл с приватным ключом

Приложение ищет сертификаты в папке, указанной аргументом командной строки, который будет описан в таблице далее по тексту.
___
# Сборка
## Windows
```bash
# Вариант 1: Использовать готовый скрипт
GenerateRelease.bat
# Вариант 2: Вручную
mkdir build && cd build
cmake ..
cmake --build . --config Release
```
___
# Запуск
Запуск происходит через командную строку с указанием или не указанием аргументов из таблицы ниже.
| Короткое имя | Длинное имя    | Описание                                               | Если не указано                            |
|:-------------|:---------------|:-------------------------------------------------------|:-------------------------------------------|
| -c           | --certificates | Задаёт путь к TLS сертификатам                         | Сервер будет запущен в незащищённом режиме |
| -p           | --port         | Задаёт значение порта, на котором будет запущен сервер | Принимает 443 или 80, если был или не был указан флаг -c/--certificates, соответственно |
| -r           | --resources    | Задаёт путь к к папке с ресурсами                      | Ищет ресурсы в той же директории, что и исполняемый файл |
